# create-ui

A simple [UIHarness](https://uiharness.com) module bootstrapper.
Create a powerful ui module using the UIHarness with a single command.

https://uiharness.com

## Usage

```bash
yarn create ui
```

or 

```bash
npx create-ui
```

then follow the prompts to generate your new module.
